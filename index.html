Bem-vindo ao Curso de GitHub

Sobre o Curso

Este curso é projetado para ensinar os fundamentos do GitHub em uma empresa, uma plataforma de hospedagem de código que permite controle de versão e colaboração.

Tópicos do Curso

  • Introdução ao Git e GitHub
  • Configuração do Ambiente
  • Trabalhando com Repositórios
  • Colaboração e Pull Requests
  • Trunk Base Development
  • Resolução de Conflitos
  • Boas Práticas
